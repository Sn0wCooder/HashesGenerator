SashaGenerator

This is a simple open source tool written in VB.NET that allows you to get MD5, SHA-1, SHA-256, SHA-384 and SHA-512 from a file.

If you want only the application, you can get it as excutable, as ZIP file or in a RAR archive.

If you want, you can deepen Secure Hash Algorithm or MD5 on Wikipedia.
